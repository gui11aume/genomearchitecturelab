Protocols



Here are the protocols used in the lab. The key steps that have been optimized are highlighted in the text. These steps can be further optimized, but we recommend following them as indicated. Wherever applicable, we indicate the reason these steps are considered crucial.

To keep a record of how protocols are used over time, we give every protocol a version number. A protocol taken from another source (in which nothing has been otpimized) is given a version number 0.5. If the protocol includes optimized steps, its version is set to 1.0. Every modification of the protocol after its publication on the website increases the version number by 0.1. Links to previous versions of the protocol (if there is any) are indicated next to the version.
